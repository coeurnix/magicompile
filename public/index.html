Prompt To Win32 App

Describe a small Windows program.

Write a short idea, let the app generate a single Win32 C file, then build and download it locally.

The final build runs in your browser using a WebAssembly build of TCC (Tiny C Compiler).

Prompt Tips

  • Keep it short and focused on one window.
  • Good fits include GUI tools, drawing, sockets, timers, multimedia, and gamepad input.
  • Avoid very large ideas or external assets.
